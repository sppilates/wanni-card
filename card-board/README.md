# card-board

Wanni가 만든 카드 게임(4×4, 총 16장) 데모입니다. HTML/JavaScript/Tailwind CSS(CDN)만 사용합니다.

## 실행

### 1) 가장 쉬운 방법
- `card-board/index.html`을 더블클릭해서 브라우저로 열기

### 2) 로컬 서버로 실행(권장)
프로젝트 루트(`커서`)에서:

```powershell
.\serve.ps1 -Port 5173
```

브라우저에서:
- `http://localhost:5173/card-board/index.html`

> 참고: Windows 환경에 따라 `serve.ps1`이 URL 예약/권한 문제로 실행 실패할 수 있습니다.
> 그 경우에는 `card-board/index.html`을 더블클릭으로 여는 방식이 가장 확실합니다.
